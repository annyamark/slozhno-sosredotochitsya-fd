# Сложно сосредоточиться

## О проекте

Адаптивный веб-сайт о проблемах концентрации внимания и способах их преодоления. Проект демонстрирует современные техники вёрстки с использованием CSS Grid и медиазапросов.

## Функциональность
- Автоматическая смена темы (светлая/темная) в зависимости от системных настроек
- Полностью адаптивный дизайн (мобильные, планшеты, десктоп)
- Кастомные элементы интерфейса
- Сложная сетка с использованием CSS Grid и subgrid

## Технологии
- HTML5
- CSS3 (Grid, Flexbox, медиазапросы, clamp(), кастомные свойства)
- Адаптивная типографика
